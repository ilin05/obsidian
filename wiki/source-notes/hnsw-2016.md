# Efficient and Robust Approximate Nearest Neighbor Search Using Hierarchical Navigable Small World Graphs

## Summary

This paper introduces HNSW, a hierarchical graph index for approximate nearest neighbor search. It builds multiple graph layers with exponentially decaying level assignment, then routes from sparse upper layers to dense lower layers for fast search.

## Key Contributions

- Proposes a fully graph-based ANN structure without requiring a separate coarse index.
- Uses hierarchy plus scale-separated links to improve routing complexity relative to prior NSW-style search.
- Uses a neighbor-selection heuristic that improves high-recall behavior and clustered-data robustness.
- Frames HNSW as conceptually similar to a probabilistic skip-list, but with proximity graph links.

## Reported Results

- Reports strong improvements over prior open-source vector-only ANN baselines in the paper's evaluation.
- Claims logarithmic complexity scaling behavior from hierarchical search design.

## Limits And Open Questions

- The paper emphasizes search speed/recall behavior, but this source alone does not resolve modern memory-tier placement questions (DRAM vs SSD vs disaggregated memory).
- Build-time and serving trade-offs versus newer large-scale systems (e.g., DiskANN/CXL-style designs) remain workload-dependent.
