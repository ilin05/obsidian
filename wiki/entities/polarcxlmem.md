# PolarCXLMem

## Profile

PolarCXLMem is a CXL-switch-based disaggregated-memory design for cloud-native databases, with an instant-recovery mechanism (PolarRecv) and CXL-aware data-sharing support.

## Why It Matters

- Moves beyond page-copying RDMA-tiered memory by pushing unified CXL memory semantics into database buffer management.
- Connects memory disaggregation with recovery and coherence design in production-oriented database architecture.
