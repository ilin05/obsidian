# Information-Theoretic Binarization For Vector Search

## Profile

This 2026 preprint argues for a binary-quantized vector search architecture that challenges HNSW + float32 + cosine as the default stack. In this vault it is mainly a speculative source on extreme compression and serverless vector-search architecture.

## Caveat

Treat broad claims cautiously. The useful durable idea is the architectural question: when does binary representation change the system design, not just the storage format?
